# Introdução ao Design System ATOM UI
O Design System do ATOM UI foi criado para ajudar a construir aplicações web e mobile com mais eficiência. Estabelecido para ser a única fonte da verdade para as equipes de produto, o Design System contém tudo, desde guias de introdução até os componentes principais destinados a serem adotados. Ele permite que as equipes de produto projetem e desenvolvam com velocidade, consistência e qualidade.

# Objetivos e benefícios

### Consistência
Crie um conjunto comum de princípios, recursos e componentes para melhorar a consistência e a experiência do usuário final em todo o portfólio de produtos digitais.

#### Benefícios
* Cria uma única fonte de verdade para as equipes de produto
* Melhora a qualidade como resultado de padrões documentados e aplicados
* Melhora a experiência do usuário como resultado de componentes, interações de design e padrões consistentes

### Eficiência
Desenvolva um sistema unificado, para aumentar a velocidade com que as equipes de produto produzem componentes novos ou aprimorados e reduzir a duplicação.

#### Benefícios
* Melhora a velocidade para designers e desenvolvedores
* Aproveita o poder coletivo de várias equipes de produto para entregar de forma mais eficaz
* Reduz a duplicação de trabalho 
* Libera tempo para se concentrar na inovação por meio de novas tecnologias e estruturas

### Colaboração
Crie uma plataforma aberta, extensa e em evolução acessível a todas as partes interessadas para apoiar a colaboração equipes.

#### Benefícios
* Melhora a colaboração entre equipes por meio de ferramentas e linguagem comuns
* Cria a base para escala, com outras equipes de produtos e mercados
* Melhora a experiência de integração para membros da equipe de produto
