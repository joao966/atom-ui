# Card

Para utilizar o componente basta realizar o import como abaixo:

```js
...
import { Card } from "@prev/ui-kit";

...
render() {
  return <Card>
    .....
    <Card.Divider/>
    outro conteudo
    <Card.Divider/>
    ...
  </Card>
}
...
```
