# FlipCard

Para utilizar o componente basta realizar o import como abaixo:

```js
...
import { FlipCard } from "@prev/ui-kit";

...
render() {
  const [flipped, setFlipped] = useState(false)

  return <FlipCard  isFlipped={flipped} onClick={() => setFlipped(!flipped)}/>
}
...
```
