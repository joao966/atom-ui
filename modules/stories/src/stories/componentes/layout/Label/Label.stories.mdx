# Label

Para utilizar o componente de Label basta realizar o import como abaixo:

```js
...
import { Label } from "atom-ui";

...
render() {
  return (
    <Label>
      Label Exemplo
    <Label/>
)}
...
```

## Exemplos

### Label Grande

O Label pode ser utilizado no tamanho grande em negrito adicionando a propriedade `size="lg"`

<Canvas>
  <Label size="lg">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Fusce finibus
    ligula sapien, et rutrum dui ultricies sit amet. Donec magna nibh, dignissim
    vel mi nec, imperdiet posuere felis.
  </Label>
</Canvas>
